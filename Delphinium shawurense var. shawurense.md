萨乌尔翠雀花(原变种)",
60a.**Delphinium shawurense var. shawurense**",

## Description
Stem mostly glabrous, hispidulous only near base. Raceme rachis glabrous. Pedicels glabrous proximally, yellow glandular puberulent distally. Sepals blue.

> * Pinus forests, streamsides; 1800--2200 m. N Xinjiang (Hoboksar Mongol Zizhixian).
