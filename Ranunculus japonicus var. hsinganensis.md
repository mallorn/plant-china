## Description
Leaf blade 3-partite, rarely 3-sect. Upper stem leaves abaxially very densely silvery sericeous-strigose. Fl. Aug.

> * Larix forests. C Nei Mongol (Bailang Shan, Horqin Youyi Qianqi).
