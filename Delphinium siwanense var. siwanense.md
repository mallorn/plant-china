细须翠雀花(原变种)",
40a.**Delphinium siwanense var. siwanense**",

## Description
Stem spreading pubescent. Pedicels spreading puberulent and yellow glandular puberulent; bracteoles 3.5--9 mm.

> * Forest margins, scrub, grassy slopes; 1900--2000 m. C and S Gansu, N Hebei, S Nei Mongol, S Ningxia, S Shaanxi, Shanxi.
  "Synonym": "&lt;I&gt;Delphinium leptopogon&lt;/I&gt; Handel-Mazzetti; &lt;I&gt;D. siwanense&lt;/I&gt; var. &lt;I&gt;leptopogon&lt;/I&gt; (Handel-Mazzetti)W. T. Wang.
