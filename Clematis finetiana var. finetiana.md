## Description
Leaves ternately 3-foliolate.

> * Sparse forests, scrub, along streams; 100--1200 m. S Anhui, Fujian, N Guangdong, N Guangxi, Guizhou, S Henan, Hubei, Hunan, S Jiangsu, Jiangxi, Shaanxi, Sichuan, Zhejiang.
