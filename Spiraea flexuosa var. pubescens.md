柔毛曲萼绣线菊",
55b.**Spiraea flexuosa var. pubescens** Liou",

## Description
Leaf blades pubescent, more densely so on veins abaxially, base rounded. Inflorescences pilose.

> Mixed forests, slopes; below 1000 m. Liaoning, Nei Mongol, Shanxi.
**曲萼绣线菊柔毛变种**

S. flexuosa Fisch. ex Cambess. var. pubescens Liou, 东北木本植物图志 288. 563. 图版102: 200. 1955.

叶片下面具短柔毛，沿叶脉较多，基部圆形，花序被稀疏柔毛。

产辽宁。生于杂木林中，海拔830米。模式标本采自临江三岔子。
